# Opal Slice 7: Macros

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add `macro` definitions, `@name` invocation, `ast...end` quasi-quoting with `$var` substitution — enough to implement `unless` macro.

**Architecture:** Macros are AST→AST template functions. `macro` stores parameter names and a body containing `ast...end` blocks. `@name expr` invokes the macro, capturing arguments as unevaluated AST (`Value::Ast`). Inside `ast...end`, `$param` is substituted with the AST bound to that parameter. The resulting AST is evaluated in place. `macroexpand` prints the expanded form without evaluating.

---

## Target program:
```opal
macro unless(condition, body)
  ast
    if !($condition)
      $body
    end
  end
end

@unless false
  print("This prints!")
end
```
Expected output: `This prints!`
