# Comments

---

## Overview

Opal supports single-line and multiline comments. Single-line comments begin with `#`, and multiline comments are delimited by `###`. There is no block-comment syntax like `/* */` or `""" """`.

---

## 1. Comment Syntax

Single-line comments begin with `#`. Multiline comments are delimited by `###`.

```opal
# This is a single-line comment

###
  This is a multiline comment.
  It can span as many lines as needed.
###

x = 42  # inline comment
```
